## 🧰 Tech Stack

| Tool | Purpose |
|------|----------|
| ⚛️ **React 19** | UI framework |
| ⚡ **Vite** | Lightning-fast bundler |
| 🎨 **Tailwind CSS + DaisyUI** | Styling and components |
| 📦 **EmailJS** | Contact form email sending |
| 🐳 **Docker** | Containerized deployment |
| ☁️ **Railway** | Hosting and automatic builds |
